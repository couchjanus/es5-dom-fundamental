Свойство parentNode

Свойство parentNode - покажет нам родительский узел элемента, к которому мы обращаемся или null для узлов, не имеющих родителя, таких как Document.Посмотрим с помощью свойства parentNode, какой у 'nav' родитель.

дочерний узел

Первый и последний дочерние узлы или null, если данный узел не имеет дочерних узлов.

  1. firstChild - первый дочерний узел элемента,
  2. lastChild - последний дочерний узел.

братские узлы

Следующий и предыдущий братские узлы. Братскими называются два узла, имеющие одного и того же родителя. Порядок их следования соответствует порядку следования в документе. Эти свойства связывают узлы в двусвязный список.

  1. nextSibling - узел после элемента,
  2. previousSibling - узел до элемента.

Свойство nodeName

Данное свойство показывает нам имя тега выбранного элемента.

Свойство nodeValue

Обратится к текстовому узлу внутри элемента можно с помощью свойства firstChild, которое вернет первый дочерний узел данного элемента - текстовый узел.

Свойство nodeType

Данное свойство может нам предоставить тип узла элемента, он вернет число, которое соответствует определенным узлам объекта Node.

  1. Число 1 - это узел элемента, например тег LI или UL,
  2. число 3 - это текстовый узел,
  3. число 8 - это комментарий.

Свойство childElementCount

Свойство Node.childElementCount предназначено только для чтения и возвращает число дочерних элементов узла.

Свойство Node.children

Свойство Node.children возвращает коллекцию (HTMLCollection) дочерних элементов узла.
